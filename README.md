# Counting fractions

## How to install

1. `https://github.com/piskun0f/counting-fractions.git`

2. `cd counting-fractions`

3. `npm i`

4. `npm run build`

5. `npm start`

6. Open Mozila Firefox in `http://localhost:3000`

## API

- Watching logs
`http://localhost:3000/logs`

- Watching users stats
`http://localhost:3000/stats`
